Variables


There are 3 main ways to declare a variable in javascript.


  • var
  • let
  • const

They each differ in functionality, and var is the oldest one of the three. Var is either function or globally scoped, which means you sometimes have access to this variable outside of its scope. This can lead to confusion or errors, and you should no longer use var anymore ever. Let is a newer version of var which is block scoped, and therefore eliminated many of the errors that can occur. Const is essentially the same as let, but you can't change the initial value of it. Except if it is an object, then you can alter the props within it.

Type Coercion


Type coercion is essentially javascript, trying to alter or convert a datatype from one thing to another behind the scenes. We as developers do not want that, as it can lead to some unexpected outcomes. One way to not have javascript do this is using === instead of == when doing conditional programming. Doing this will force javascript to also validate the data types aswell.

Functions


you can create functions in javascript and call them at a later point, just like most programming languages. There are multiple ways to create functions in javascript. function declarations (hoisted), function expressions (not hoisted) and arrow functions. Functions can take in parameters that can be used inside the function scope. This can be all the datatypes we covered earlier and even other functions creating a callback function.
